# GUI Stopwatch (Python)

## About

A simple GUI Stopwatch built using Python and Tkinter.
This application allows users to measure elapsed time with start, stop, and reset controls.

---

## ✨ Features

✔️ Start stopwatch
✔️ Stop stopwatch
✔️ Reset stopwatch
✔️ Real-time timer display
✔️ Simple and user-friendly GUI

---

## 🛠️ Tech Used

* Python
* Tkinter

---

## 🚀 How to Run

1. Download the project
2. Open the Python file
3. Run the program
4. Use Start, Stop, and Reset buttons
